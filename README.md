# Mobile and Wireless Communications Security

Coursework repository for **Mobile and Wireless Communications Security** (Ασφάλεια Κινητών και Ασύρματων Επικοινωνιών).
Includes smart contracts & DApps, a TLS/HTTPS Node.js challenge (cert generation + server), and security writeups (forensics, incident response).

> Educational use only — not production-audited. Do not use to store real personal data on-chain.
